# Individual-Education-Trajectory
University Project
```mermaid
graph LR
    subgraph Внешняя среда
        A[Абитуриенты]
        B[Студенты]
        C[РОПы]
        E[Внешние системы]
    end

    subgraph Система
        subgraph Фронтенд
            Z1[Веб-интерфейс]
        end
        subgraph Бэкенд
            Z2[API сервиса ИОТ]
            Z3[Логика формирования ИОТ]
            Z4[Хранилище данных]
        end
    end

    subgraph База данных
        F{Курсы}
        G{Компетенции}
        H{Профессии}
    end

    A --Выбор профессии--> Z1
    B --Выбор профессии--> Z1
    C --Изменение курсов--> Z1
    D --Взаимодействие с веб-интерфейсом--> Z1
    Z1 --Получение курсов--> A
    Z1 --Получение ИОТ--> B
    Z1 --Новые курсы--> C
    E --> Z2

    Z1 --Запрос на сервер--> Z2
    Z2 --Запрос формирования ИОТ--> Z3
    Z3 --Запрос данных из хранилища--> Z4
    Z4 --Получение данных--> Z2
    Z2 --Вывод данных--> Z1

    Z3 --Поиск профессии в таблице--> H
    H --Поиск общих связей--> G
    G --Поиск общих связей--> F
    F --Получение рекомендуемых курсов--> Z2

    Z2 --Вывод на экран--> E
```
